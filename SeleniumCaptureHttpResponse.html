Example Web Page for SeleniumCaptureHttpResponse

This page is an example for the SeleniumCaptureHttpResponse GitHub repository.

It demonstrates how Selenium WebDriver and ChromeDriver can capture HTTP responses from JavaScript API calls on third‑party sites. This can be useful for testing, or as part of scraping 3rd party websites.

The JavaScript in this web page makes two API calls that return JSON, and some of that data is shown below. The repository includes a .NET 8 console app that captures the full JSON responses and writes them to the console.
