Example Web Page for BrowserCaptureRewrite

This page is an example for the samples project in the BrowserCaptureRewrite GitHub repository.

BrowserCaptureRewrite is a NuGet package that facilitates the interception, capture, and rewriting of HTTP requests/responses performed on a web page using Playwright. The Metalhead.BrowserCaptureRewrite.Samples project contains various examples demonstrating how the NuGet package can be used to capture and manipulate HTTP traffic.

The JavaScript in this web page fetches bands_a-m.json, bands_n-z.json, and albums.json, then displays below. albums.json is fetched after a delay to demonstrate the importance of providing a capture completion predicate for capturing delayed responses.
