The purpose of this web page is to act as an example web page for the the samples project in my BrowserCaptureRewrite repository on GitHub.

BrowserCaptureRewrite is a NuGet package that facilitates the interception, capture, and rewriting of HTTP requests/responses performed on a web page using Playwright. The samples project contains various examples demonstrating how the NuGet package can be used to capture and manipulate HTTP traffic.

The JavaScript in this web page makes two calls, which both return JSON. The data returned from those calls are displayed below.
